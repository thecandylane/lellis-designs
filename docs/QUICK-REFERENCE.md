# L. Ellis Designs - Quick Reference Card

## Your Daily Workflow

```
1. Check email for new orders
2. Log into lellisdesigns.com/admin
3. Process orders:
   - Make the buttons
   - Mark as "Ready" (pickup) or "Shipped" (UPS)
4. Customer gets automatic email notification
```

---

## Admin Panel Cheat Sheet

| I want to... | Go to... |
|--------------|----------|
| See new orders | Admin → Orders |
| Add a button | Admin → Buttons → Create New |
| Add a category | Admin → Categories → Create New |
| See custom requests | Admin → Custom Requests |
| Change site colors | Admin → Site Settings |

---

## Order Actions

| Order Type | When Done | What to Click | What to Enter |
|------------|-----------|---------------|---------------|
| **Pickup** | Buttons ready | "Mark Ready" | Pickup address |
| **UPS** | Package shipped | "Mark Shipped" | Tracking number |
